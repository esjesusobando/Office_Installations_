# Product Manager — Skill Index

## Descripción
Gestión de producto: brainstorming, planificación técnica, ejecución de planes, gestión de épicas/tareas en Jira y escritura de documentos estratégicos.

## Sub-Skills

| #   | Skill                       | Descripción                                    |
|-----|-----------------------------|------------------------------------------------|
| 01  | `01_Brainstorming`          | Sesiones estructuradas de ideación de producto |
| 02  | `02_Technical_Planning`     | Planificación técnica de features              |
| 03  | `03_Executing_Plans`        | Ejecución de planes de producto                |
| 04  | `04_Jira_Epic`              | Creación de épicas en Jira                     |
| 05  | `05_Jira_Task`              | Creación de tareas en Jira                     |
| 06  | `06_Branch_Pr`              | Workflow de branch y PR para producto          |
| 07  | `07_Writing_Strategy_Memos` | Escritura de memos estratégicos                |
| 08  | `08_Prd`                    | Generación de Product Requirements Documents   |

## Uso
Usar en contextos de planificación y gestión de producto.
