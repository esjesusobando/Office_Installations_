# Product Design — Skill Index

## Descripción
Diseño de producto y sistemas visuales: desde taste-skill y diseño minimalista hasta brand identity, design systems y herramientas de diseño especializadas.

## Sub-Skills

| # | Skill | Descripción |
|---|-------|-------------|
| 01 | `01_Taste_Skill` | Diseño de alto impacto visual con criterio estético |
| 02 | `02_Soft_Skill` | Diseño suave y accesible |
| 03 | `03_Minimalist_Skill` | Diseño minimalista funcional |
| 04 | `04_Redesign_Skill` | Metodología de rediseño de interfaces |
| 05 | `05_Output_Skill` | Generación de outputs de diseño |
| 06 | `06_Dieter_Rams_Design` | Principios de diseño según Dieter Rams |
| 07 | `07_Brand_Identity` | Construcción de identidad de marca |
| 08 | `08_Brand_Voice_Generator` | Generación de voz de marca |
| 09 | `09_Canvas_Diagram_Studio` | Diagramas y canvas estratégicos |
| 10 | `10_Visual_Language` | Sistema de lenguaje visual |
| 11 | `11_Pencil_Design_Studio` | Sketching y prototipado |
| 12 | `12_Shadcn` | Componentes Shadcn/UI |

## Uso
Cargar `01_Taste_Skill` como base para cualquier tarea de diseño visual. Complementar con skills específicas según el contexto.
