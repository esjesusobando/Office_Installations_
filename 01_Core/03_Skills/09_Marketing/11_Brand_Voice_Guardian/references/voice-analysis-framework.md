# Voice Analysis Framework

## Las 6 Dimensiones del Análisis de Voz

### 1. Ritmo de las Oraciones

**Qué observar:**
- Longitud promedio (cortas < 10 palabras / medianas 10-20 / largas > 20)
- Patrón de variación (monótono / mixto / intencional)
- Uso de fragmentos ("Así, sin más." / "Y me cayó el twenty.")
- Puntuación que usa (comas, guiones, puntos suspensivos, saltos de línea)

**Cómo documentar:**
```
RITMO: [Corto y contundente / Largo y fluido / Mixto]
VARIACIÓN: [Alta / Media / Baja]
FRAGMENTOS: [Frecuentes / Ocasionales / Ninguno]
```

### 2. Huella de Vocabulario

**Qué observar:**
- Palabras que usa 2+ veces (favoritas)
- Palabras que NUNCA usa (ausencias)
- Nivel léxico (casual / técnico / mixto)
- Coloquialismos / regionalismos

**Cómo documentar:**
```
KEYWORDS: [palabra1, palabra2, palabra3...]
NUNCA: [palabra1, palabra2...]
NIVEL: [Casual / Técnico / Mixto]
```

### 3. Temperatura Emocional

**Qué observar:**
- Emoción dominante (curiosidad / empatía / urgencia / calidez / frustración / orgullo)
- Ratio lógica vs. emoción (más lógica / más emoción)
- Vulnerabilidad (superficial / moderada / profunda)

**Cómo documentar:**
```
EMOCIÓN: [Primaria] + [Secundaria]
VULNERABILIDAD: [Alta / Media / Baja]
```

### 4. Punto de Vista (PDV)

**Qué observar:**
- Ratio "yo" vs "tú" (habla de sí mismo / del lector / equilibrado)
- Posicionamiento (amigo / guía / maestro / compañero)

**Cómo documentar:**
```
PDV: [Amigo a amigo / Guía / Maestro / Compañero]
YO-TÚ: [Ambas cerca / Yo cerca, tú lejos / Tú cerca, yo lejos]
```

### 5. Hábitos Estructurales

**Qué observar:**
- Cómo abre (pregunta / afirmación / historia / claim audaz)
- Cómo cierra (CTA / línea reflexiva / pregunta al lector)
- Formato preferido (listas / prosa / mixto)

**Cómo documentar:**
```
ABRE: [Tipo de apertura habitual]
CIERRA: [Tipo de cierre habitual]
FORMATO: [Lists / Prose / Mix]
```

### 6. Lo Que Nunca Dice

**Qué observar:**
- Frases que feel out of brand
- Temas que evita
- Tonos que no le pegan

**Cómo documentar:**
```
NUNCA: [frase1, frase2...]
EVITA: [tema1, tema2...]
TONOS: [fuera: tono1, tono2]
```

## Tarjeta de Voz Template

```
TARJETA DE VOZ — [Nombre / Handle]

RITMO: 
VOCABULARIO: 
EMOCIÓN: 
PDV: 
ESTRUCTURA: 
NUNCA DIGAS: 
SIEMPRE SUENA A: 
```

---

## Ejemplo de Tarjeta Completada

```
TARJETA DE VOZ — @juancreator

RITMO: Oraciones cortas. Fragmentos bienvenidos. Ocasional línea más larga para dar profundidad.
VOCABULARIO: Palabras simples. Sin tecnicismos. "Real", "honesto", "entender", "nomás"
EMOCIÓN: Empatía cálida. Personal pero sin drama. Relatable sobre inspiracional.
PDV: Amigo a amigo. Nunca maestro. "Yo" y "tú" siempre cerca.
ESTRUCTURA: Gancho audaz → momento real → 1 idea clara → cierre tranquilo
NUNCA DIGAS: "potenciar", "empoderar", "en el mundo actual", "la mayoría no sabe"
SIEMPRE SUENA A: Un mensaje de WhatsApp de un amigo que descubrió algo
```
