# Project Manager — Skill Index

## Descripción
Gestión del ciclo de vida de proyectos: rituales diarios, procesamiento de backlog, reviews semanales y generación de contenido. Orientado a proyectos de software con metodologías ágiles.

## Sub-Skills

| # | Skill | Descripción |
|---|-------|-------------|
| 01 | `01_Morning_Standup` | Ritual de apertura diaria — qué hice, qué hago, bloqueantes |
| 02 | `02_Backlog_Processing` | Procesar y priorizar el backlog |
| 03 | `03_Weekly_Review` | Review semanal de progreso y ajuste de prioridades |
| 04 | `04_Sunday_Ritual` | Ritual dominical de planificación de semana |
| 05 | `05_Best_Practices` | Best practices de gestión de proyectos |
| 06 | `06_Finishing_A_Development_Branch` | Proceso de cierre y merge de una rama |
| 07 | `07_Running_Tests` | Workflow de ejecución y validación de tests |
| 08 | `08_Content_Generation` | Generación de contenido para el proyecto |

## Uso
Activar la sub-skill correspondiente según el momento del ciclo de proyecto.
