# 🚀 Demos Junior — Claude Computer Use

> Tutoriales paso a paso

---

## Demo 01: Instalar Claude Desktop + Computer Use

### Paso 1: Descargar
1. Vas a https://claude.ai/download
2. Download para Mac
3. Instalar la app

### Paso 2: Login
1. Abrir Claude Desktop
2. Login con tu cuenta (debe ser Pro o Max)

### Paso 3: Activar Computer Use
1. Click en tu nombre (bottom left)
2. Settings
3. Desktop app → General
4. Toggle "Browser Use" → ON
5. Toggle "Computer Use" → ON
6. Aprobar permisos del sistema

**Tiempo: 5 minutos**

---

## Demo 02: Usar Cowork Mode

### Paso 1: Entrar
1. En Claude Desktop, ver 3 tabs arriba: Chat, Cowork, Code
2. Click en "Cowork"

### Paso 2: Primera tarea simple
```
"Open Xcode and create a new project"
```

### Paso 3: Observar
- Claude toma screenshots
- Analiza qué hay en pantalla
- Clickea, typing, navigating
- Repite hasta completar

**Tiempo: 2 minutos**

---

## Demo 03: Configurar Dispatch (desde teléfono)

### Paso 1: Habilitar
1. Settings → Cowork
2. Toggle "Dispatch" → ON

### Paso 2: Setup QR
1. Click "Dispatch" en sidebar
2. Escaneá el QR code con tu phone

### Paso 3: Usar desde phone
1. Abrir Claude mobile app
2. Ir a Dispatch tab
3. Escribir tarea
4. Claude la hace en tu Mac remoto

**Ejemplo:**
```
"Go to my Downloads folder and organize all PDFs into a folder called Work"
```

**Tiempo: 3 minutos**

---

## Demo 04: Conectar Notion MCP

### Paso 1: Instalar
```bash
npx -y @notionhq/notion-mcp-server
```

### Paso 2: Configurar en Claude Desktop
1. Settings → Integrations
2. Buscar "Notion"
3. Agregar API key

### Paso 3: Usar
```
"Create a new page in Notion about my meeting notes"
```

**Tiempo: 5 minutos**
