# Prompts Usados — IA Generativa (ES)

## Prompts Originales del Contenido

### Prompt Básico Midjourney
```
[Sujeto] en [contexto], [estilo], [iluminación], [detalles]
```

### Prompt Avanzado Midjourney
```
[Sujeto] + [contexto] + [estilo específico] + iluminación [tipo] + [parámetros técnicos]
--ar [relación aspecto]
--v [versión]
--stylize [valor]
--quality [calidad]
```

---

## Prompts para 7 Perfiles (ES)

### 01. Project Manager
```
Eres un Project Manager en una agencia creativa. 
Usa IA generativa para planificar sprints de contenido visual.
Crea un calendario de 4 semanas donde:
- Semana 1: Experimentación con Midjourney
- Semana 2: Creación de assets para campaña
- Semana 3: A/B testing de imágenes
- Semana 4: Entrega final optimizada

Cada deliverable debe incluir: prompt usado, resultado, learnings.
```

### 02. Product Manager  
```
Eres un Product Manager de una tool de IA generativa.
Define la estrategia de producto para lanzar una nueva feature de generación de imágenes.
Incluye:
- Problema que resuelve
- Caso de uso principal
- Métricas de éxito
- Roadmap提议
- Competencia vs diferenciación
```

### 03. Product Design
```
Eres un Product Designer creando un sistema de diseño para una app de IA generativa.
Diseña el flujo de usuario para:
- 输入 de prompt por usuario
- Selección de estilo visualizar
- Preview y refinamiento
- Exportación de imagen

Incluye: wireframes, user journey, decisiones de diseño.
```

### 04. Vibe Coding
```
Eres un developer usando IA para coding + diseño.
Crea un prompt que genere:
- Componentes UI para un dashboard
- Imágenes para landing page
- Iconos y illustrations
- Mockups de app

 Código example:
/imagine prompt: [descripción del componente], ui design, clean minimal, figma style
```

### 05. Testing
```
Eres un QA tester validando outputs de IA generativa.
Crea un test plan para MIDJOURNEY:
- Validación de prompts (sintaxis正确)
- Consistencia de estilos
- Resolución y calidad
- Cumplimiento de políticas

Incluye: casos de prueba, expected results, criterios de aceptación.
```

### 06. DevOps
```
Eres un DevOps integrando IA generativa en un pipeline de deployment.
Diseña un workflow donde:
1. Developer escribe código
2. AI genera imágenes automáticamente
3. Pipeline verifica calidad
4. Deploy a producción

Usa APIs de: Stability AI, DALL-E, o Midjourney.
```

### 07. Marketing
```
Eres un Growth Marketer usando IA generativa para campañas.
Crea una campaña completa:
- 10 prompts para diferentes fases del funnel
- Adaptación por canal (IG, FB, X, LinkedIn)
- A/B testing de imágenes
- ROI tracking

Cada prompt debe incluir: copy + imagen + hashtag + CTA.
```
