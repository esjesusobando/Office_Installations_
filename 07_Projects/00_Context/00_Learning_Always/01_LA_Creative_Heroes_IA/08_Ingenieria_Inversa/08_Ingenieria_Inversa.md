# Ingeniería Inversa: Creative Heroes Workshop

## Análisis: Cómo fue construido el contenido

### Tecnologías usadas

| Tecnología         | Propósito            | Cuándo                    |
|--------------------|----------------------|---------------------------|
| **YouTube**        | Hosting del video    | 2024-presente             |
| **Zoom/Streaming** | Clases en vivo       | Durante el workshop       |
| **Midjourney**     | Demo de imágenes     | Ejemplos prácticos        |
| **ChatGPT**        | Asistencia general   | Soporte                   |
| **Leonardo.ai**    | Alternativa gratuita | Para usuarios sin Discord |
| **Discord**        | Comunidad + soporte  | Feedback y comunidad      |

### Patrones de diseño del contenido

| #   | Patrón                     | Descripción                            |
|-----|----------------------------|----------------------------------------|
| 1   | **Hook emocional**         | "La IA no va a reemplazarte"           |
| 2   | **Demo primero**           | Ejemplos visuales antes de teoría      |
| 3   | **Framework transferable** | De creativo → agencia                  |
| 4   | **Escala social**          | "+600K estudiantes" como prueba social |
| 5   | **CTA claro**              | "Regístrate gratis"                    |

### Estructura del contenido

```
1. HOOK (30 seg): Problema/ Oportunidad
2. CONTEXTO (2 min): Por qué la IA importa
3. DEMO (5 min): Midjourney en acción
4. ESTRUCTURA (3 min): Comandos y parámetros
5. CASO DE USO (3 min): De creativo a agencia
6. CTA (1 min): Siguiente paso
```

### Qué podemos aprender

| #   | Learning                     | Aplicación                        |
|-----|------------------------------|-----------------------------------|
| 1   | Criterio > Herramienta       | Enfoque en visión, no en tool     |
| 2   | Demo primero, teoría después | Siempre mostrar antes de explicar |
| 3   | Escala social funciona       | "+X usuarios" genera confianza    |
| 4   | Comunidad como asset         | Discord/grupos crean retention    |
| 5   | De 0 a agency                | Modelo de negocio claro           |

### Cómo replicar enfoques similares

#### Para Workshops:
```
1. Hook con tensión (problema o oportunidad)
2. Demo live de la herramienta
3. Estructura simple (3-5 puntos)
4. Caso de éxito como prueba
5. CTA claro al final

Duración óptima: 30-60 minutos
```

#### Para contenido educativo:
```
1. Qué vas a aprender (outcome claro)
2. Por qué importa (contexto)
3. Cómo hacerlo (pasos)
4. Ejemplo práctico (demo)
5. Ejercicio para practicar
6. Siguiente paso (CTA)
```

#### Para marketing de cursos:
```
1. Problema del audiencia
2. Solución (tu curso)
3. Prueba social (estudiantes, resultados)
4. Diferenciador (qué te hace único)
5. Urgencia (fechas, cupos)
```
