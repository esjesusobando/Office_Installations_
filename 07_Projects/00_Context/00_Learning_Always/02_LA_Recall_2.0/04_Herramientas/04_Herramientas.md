## Workflows Posibles

### Save → Chat → Quiz → Graph
```
1. Guardá contenido (extension)
2. Chateá para entender (AI chat)
3. Generá quiz para estudio (Quiz 2.0)
4. Visualizá conexiones (Graph View)
```

### Save → Connect → Automate
```
1. Guardá contenido
2. Creá conexiones manuales
3. Conectá vía MCP a tus AI tools
4. Automatizá flujos con webhooks
```

## 🎯 Recomendación para PersonalOS

Integración ideal con **Engram** + **GGA**:

- Engram = memoria de sesión + contexto

- GGA = code review sobre automation

*Stack: Engram → sessions, Recall → knowledge, GGA → quality*
