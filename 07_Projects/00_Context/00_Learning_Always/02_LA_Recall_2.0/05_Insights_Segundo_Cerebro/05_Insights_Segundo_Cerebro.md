## 🔑 Insight 01: AI Niveló el Campo

**El contexto:**
Todos tenemos acceso a los mismos modelos de AI. La diferencia ya no es el modelo — es el KNOWLEDGE.

**Qué significa:**
- Notion/Obsidian guardan DATOS
- Recall guarda KNOWLEDGE + AI que lo entiende
- Tu ventaja: contexto personalizado

**Aplicación:**
Usar Recall para construir un "second brain" que tu AI puede consultar.

---

## 🔑 Insight 02: Chat Dual Mode

**El contexto:**
Recall te permite elegir entre chatear con:
1. Solo tu knowledge
2. Solo internet
3. Ambos simultáneamente

**Qué significa:**
- Cuando necesitás precisión → solo tu knowledge
- Cuando necesitás actualizar → internet
- Cuando necesitás ambos → modo dual

**Aplicación:**
- "¿Qué guardé sobre X?" → knowledge only
- "¿Qué dijo el speaker en este video?" → knowledge
- "¿Cuál es el estado actual de AI en 2026?" → internet

## 🔑 Insight 03: Spaced Repetition Real

**El contexto:**
Quiz 2.0 tiene spaced repetition real con 5 stages:
New → Learning → Practiced → Confident → Mastered

**Qué significa:**
- No es solo "repetir" — es scheduling inteligente
- Intervalos: 1 día hasta 3 meses
- Incorrectas bajan a Learning para refuerzo

**Aplicación:**
Guardar contenido importante = generar quiz inmediatamente.

## 🔑 Insight 06: De Consumo Pasivo a Aprendizaje Activo

**El contexto:**
El shift mental más importante: de "guardar y olvidar" a "guardar, conectar, recordar".

**Qué significa:**
- Guardar sin revisar =浪费时间
- Guardar + quiz + graph = aprendizaje compuesto
- El 80/20: 20% esfuerzo, 100% retención

**Aplicación:**
Por cada contenido guardado → generar quiz + crear conexión.
