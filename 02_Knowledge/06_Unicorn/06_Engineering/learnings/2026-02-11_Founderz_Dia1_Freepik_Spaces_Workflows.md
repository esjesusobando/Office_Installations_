## Resumen Ejecutivo

El webinar de Founderz introduce el concepto de **flujos de trabajo basados en nodos** como el nuevo estándar profesional para la creación de contenido visual con IA. La shift más importante: pasar de herramientas aisladas a sistemas automatizados y reutilizables.

---

## Conceptos Fundamentales

### 1. ¿Por qué Nodos?

**Problema anterior:**
- 5+ apps desordenadas = caos creativo
- Pérdida de tiempo entre herramientas
- Versiones sin control
- Inconsistencia en resultados

**Solución nodos:**
- Un solo lienzo visual con nodos conectados
- Todo visible, coherente y replicable
- Del producto base a 11 activos en minutos

### 2. Anatomía de un Nodo

Los nodos funcionan como una **carrera de relevos creativa**:
- **Entradas a la izquierda** — datos que recibe
- **Salidas a la derecha** — datos que envía
- El "testigo" (datos) fluye de nodo en nodo

### 3. Tipos de Nodos en Freepik Spaces

| Nodo | Función |
|------|---------|
| **Texto** | Escribir prompts, notas, comentarios |
| **Imagen** | Imágenes de referencia, entradas |
| **Video** | Contenido de video base |
| **Asistente** | IA que expande ideas, genera texto, transforma prompts |
| **Generador de Imagen** | Convierte texto/referencias en imágenes |
| **Generador de Video** | Crea video a partir de prompts o imágenes |
| **Mejorar Calidad (Upscale)** | Aumenta resolución con Magnific |
| **Lista** | Agrupar múltiples elementos |
| **Variaciones** | Generar variantes controladas |

## Flujo de Trabajo: "Máquina de Contenido"

### Paso 1: Preparar el activo base
- Imagen base del producto/modelo
- Esta imagen actúa como nodo de referencia

### Paso 2: Idea Principal (Nodo de Texto)
- Storytelling: quiénes son los personajes
- Qué emoción se quiere transmitir

### Paso 3: Asistente AI
- Conecta idea e imagen base
- El asistente lee ambas, analiza la imagen
- Identifica elementos clave para mantener consistencia

### Paso 4: Prompt Mejorado
- El Asistente transforma la idea en prompt estructurado
- Añade parámetros técnicos (estilo, iluminación, composición)

### Paso 5: Generación Consistente
- Conectar prompt mejorado + referencia visual al generador
- Mantener consistencia en personajes y estructura

### Paso 6: Variantes y Formatos
- Generar 1-2 variaciones por color/acabado
- Elegir modelo (Nano Banana, C-Dream) y aspect ratio
- 1:1 para feed, 9:16 para Reels/Stories

### Paso 7: Escalado y Exportación
- Duplicar la rama "Escenas" y cambiar solo 2-3 parámetros
- Renombrar por bloques: `Base → Variantes → Escenas → Multi → Video`

## Consistencia de Personajes/Productos

**Punto clave:** Todas las imágenes deben estar conectadas directamente con la imagen base original.

El flujo de trabajo organizado por nodos permite obtener resultados consistentes:
1. Imagen Base → nodo de referencia inicial
2. Idea Principal → storytelling
3. Asistente AI → analiza imagen y crea prompts
4. Prompt Mejorado → conecta con generador
5. Generador → mantiene consistencia
