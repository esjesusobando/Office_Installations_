# 📚 Contexto y Fundamentos

## 1.1 ¿Qué es la Inteligencia Artificial Generativa?

La IA generativa es un campo de la inteligencia artificial que se centra en **crear contenido nuevo** a partir de patrones aprendidos durante el entrenamiento con grandes cantidades de datos. A diferencia de los sistemas de IA tradicionales que clasifican o analysen datos, la IA generativa Produce:

| Tipo de contenido | Descripción |
|-----------------|-------------|
| **Texto** | Escritos, artículos, código, respuestas conversacionales |
| **Imagen** | Fotografías, ilustraciones, arte digital, gráficos |
| **Vídeo** | Contenido en movimiento, animaciones |
| **Audio** | Música, voces sintéticas, efectos de sonido |

# 🎯 Técnicas de Prompt Engineering

## 2.1 Fundamentos del Prompting

El **prompt engineering** es el arte y la ciencia de escribir instrucciones efectivas para obtener resultados específicos de los modelos de IA. No basta con pedir cosas de forma vag; es necesario:

1. **Serclaro y específico**: Expresar exactamente lo que quieres
2. **Añadir contexto**: Proporcionar información relevante de fondo
3. **Definir el estilo**: Indicar el estilo visual o interpretativo deseado
4. **Establecer el propósito**: Tener un fin estratégico definido

## 2.2 Estructura de un Prompt Profesional

```
[Sujeto] + [Contexto] + [Estilo] + [Propósito]
```

### Ejemplo de Prompt Básico vs. Profesional

| Nivel | Prompt | Resultado |
|-------|-------|----------|
| **Básico** | "Una mujer" | Imagen genérica, sin dirección clara |
| **Intermedio** | "Una mujer con flores doradas" | Mejor pero todavía impreciso |
| **Profesional** | "Una mujer caucásica con un pequeño toque dorado en la cara, con alguna mariposa, estilo editorial de revista Vogue, iluminación de estudio, colores vibrantes, fondo desenfocado, para uso en publicidad de marcas de lujo" | Imagen de alta calidad lista para uso comercial |

## 2.3 Técnicas Avanzadas

### A. Especificidad Extreme

| En lugar de... | Usa... |
|---------------|--------|
| "un gato" | "un gato azul siamés con ojos verdes esmeralda, pelaje sedoso, orejas puntiagudas, expresión curioso, estilo fotográfico, luz natural dorada de atardecer, fondo borroso de jardín" |

### B. Referencias de Estilo

| Reference | Ejemplo de Prompt |
|-----------|-----------------|
| **Artista** | "estilo de Vincent van Gogh", "como Picasso" |
| **Publicación** | "como fotografía de revista Vogue", "estilo editorial de GQ" |
| **Estudio** | "estilo Studio Ghibli", "como película de Pixar" |
| **Época** | "estilo años 50", "aesthetic retro" |

### C. Control de Iluminación

- "iluminación de estudio profesional"
- "luz natural dorada de atardecer"
- "backlight suave"
- "claroscuro dramático"
- "rim lighting"

### D. Definir Composición

- "primer plano (close-up)"
- "plano medio"
- "áng desde abajo (low angle)"
- "vista aérea (bird's eye)"
- "composición triangular"

### E. Iteración y Refinamiento

> *"No te conformes con el primer resultado, construye sobre él."*

El proceso es:
1. Generar una primera versión
2. Analizar qué funciona y qué no
3. Añadir o modificar instrucciones
4. Regenerar y refinar
5. Repetir hasta obtener el resultado deseado
